Spec -> testes

Comandos cypress -
	cy.visit('') => ir para a url
	cy.get('') => #id
			.class
			('tag[atributo="atributeName"]')
	cy.contains => contém / validação
	cy.should => asserções / validação
	seletor do cypress => data-cy

	it.only => roda somente esse teste
	it.skip => skip o teste
